## Запуск тестів

### Всі UI тести
```bash
nvim --headless --noplugin -u tests/minimal_init.lua -l tests/run_ui_tests.lua
```

### Або разом з усіма тестами
```bash
./test.ps1
```

## Примітки

Тести перевіряють тільки структуру API і базову функціональність без vim.api, 
оскільки headless mode не підтримує повноцінні vim API операції.

Для тестування реальної функціональності з UI використовуйте ручне тестування в Neovim.
