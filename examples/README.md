# Examples

Приклади конфігурацій для nvim-agent.

## example_config.lua

Повний приклад конфігурації з усіма можливими опціями та коментарями.

**Використання:**

```lua
-- У вашому init.lua або конфігурації плагіна
require('nvim-agent').setup({
    -- Скопіюйте потрібні частини з example_config.lua
})
```

**Швидкий старт:**

```bash
# Подивіться приклад
nvim examples/example_config.lua
```
